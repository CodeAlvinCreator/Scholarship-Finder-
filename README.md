 Scholarship-Finder-
Scholarship Finder

A lightweight and easy-to-use web application designed to help African students quickly discover scholarship opportunities. Built with **HTML**, **CSS**, and **JavaScript**, this project runs entirely in the browser — no backend or server setup required.

---

Overview

Scholarship Finder was created to simplify access to scholarship information for African students. Many students struggle to find updated, reliable, and consolidated scholarship resources. This tool offers a clean interface where users can browse available scholarships and filter based on the details provided.

---
Features

Simple and clean user interface
Fast — runs directly from your browser
No installation or dependencies required
Easy to edit — scholarships can be added manually in JavaScript
Fully open-source and customizable
